Welcome to the Todo App

Introduction

This is an auth todo application built with Next.js, GraphQL, Apollo Server, and MongoDB.

Features

  • Add new todos
  • Mark todos as complete or incomplete
  • Delete todos
  • View list of todos

Technologies Used

  • Next.js
  • GraphQL
  • Apollo Server
  • MongoDB
  • Json Web Token

Installation

          
git clone https://github.com/thefutureseer/todo-app.git
cd todo-app
npm install
npm run dev

Open your browser and navigate to http://localhost:3000 to view the app.

Usage

Follow the instructions on the homepage to add, mark, delete, and view todos.
